# 数字表示

本书将大量使用以下方法对数字进行简写：

## 计数法

科学和工程计数法可用于将长数字简写为与 10 的指数相乘的形式：

| 数字      | 科学计数法          | 工程计数法 |
| --------- | ------------------- | ---------- |
| 1,000,000 | $1.0 \times 10^{6}$   | 1.0E6        |
| 1,300,000 | $1.3 \times 10^{6}$ | 1.3E6      |
| 0.001     | $1.0 \times 10^{-3}$  | 1.0E-3       |

## 国际单位制前缀

还可以将长数字通过国际单位制词头进行缩写，以下为常用符号：

| 符号  | 英文名称 | 中文名称 | 对应大小   |
| ----- | -------- | -------- | ---------- |
| f     | fento    | 飞       | $10^{-15}$ |
| p     | pico     | 皮       | $10^{-12}$ |
| n     | nano     | 纳       | $10^{-9}$  |
| u     | micro    | 微       | $10^{-6}$  |
| m     | milli    | 毫       | $10^{-3}$  |
| k / K | kilo     | 千       | $10^{3}$   |
| M     | mega     | 兆       | $10^{6}$   |
| G     | giga     | 吉       | $10^{9}$   |
| T     | tera     | 太       | $10^{12}$  |
| P     | peta     | 拍       | $10^{15}$  |

例子：

| 原数字      | 简写形式 |
| ----------- | -------- |
| 1,500,000   | 1.5M     |
| 210,500,000 | 210.5M   |
