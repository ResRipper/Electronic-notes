# 触电

人体作为一导体，在两侧施加电压差就会有电流产生。随着电流大小增加，会对人体产生不同程度的效果：

| 60Hz 下电流（mA） | 效果                       |
| ----------------- | -------------------------- |
| 1                 | 几乎无法感知               |
| 16                | 人体承受及脱手极限         |
| 20                | 呼吸肌瘫痪                 |
| 100               | 房颤阈值                   |
| 2k                | 心脏骤停和内脏损伤         |

## 安全阈值

对人体造成伤害的主要是流经的电流，电流大小的部分影响因素如下：

- 人体电阻随接触点及个体不同而变化，在皮肤湿润或干燥时会进一步影响电阻值
- 高电压会破坏外层皮肤，导致人体电阻骤降
- 接触时间和电流频率会导致不同效果

因此无法界定精确的安全电压值，各标准定义的值也不完全一致。教科书上称 `36V` 为安全电压，但对应标准早已过期，替代的 `GB/T 3805-2008` 标准对电压值进行了更加详细的分级，另外 [IEC 60479-1](https://en.m.wikipedia.org/wiki/File:IEC_TS_60479-1_electric_shock_graph.svg) 也对接触时间和电流大小与损伤的关联制作了一系列图表。

如果懒得查标准，一般可认为 `12V` 及以下为绝对安全电压。

## 防护方法

- 避免带电操作，预先使用电笔或万用表的 NCV 档/电压档确保已断电
- 对设备进行接地，以确保人体非到地的最低电阻路径
- 保持体表干燥
- 佩戴绝缘手套进行操作
- 穿戴胶鞋或踩在其它绝缘物上进行操作
- 避免形成穿过心脏的回路
