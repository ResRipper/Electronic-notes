# 基本原则

PCB 布线必须先满足工艺参数，否则将无法生产。以下为一些常见厂商的 PCB 工艺参数页：

- [嘉立创（JLC）](https://www.jlc.com/portal/vtechnology.html)
- [华秋（NextPCB）](https://www.hqpcb.com/ability.html)
- [Aisler](https://community.aisler.net/t/pcb-portfolio/101)
- [PCBWay](https://www.pcbway.com/capabilities.html)

下面为一些比较通用的参数：

| 项目         | 限制（mm）    |
| ------------ | ------------- |
| 板厚         | 0.8，1.0，1.6 |
| 线宽         | ≧ 0.2         |
| 线边到边距离 | ≧ 0.1         |
| 过孔内径     | ≧ 0.3         |
| 过孔外径     | ≧ 0.5         |
| 孔边到铜距离 | ≧ 0.1         |

以下为一些需要注意的点：

- 很多厂商会给对制造能力进行价格分段，需要到计价页面才能看到收费标准。
- **不要**贴着厂商能力进行设计，会潜在导致良率降低。
- 设计完成后，还建议使用厂商提供的可制造性（DFM）工具进行检查，并进行相应修改。

如果需要进行贴片生产，则还需要参考厂商对于 SMT 的工艺限制。

## 分级

IPC 2221 标准制定了一系列电路板设计规范，并进行了以下分级，本章节中也将提到这些等级：

| 性能等级 | 描述                                     |
| -------- | ---------------------------------------- |
| Class 1  | 一般电子产品，寿命有限，能用就行         |
| Class 2  | 特定用途产品，对寿命有一定要求，尽量耐用 |
| Class 3  | 性能和寿命敏感产品，不允许出现问题       |

| 可制造性等级 | 描述         |
| ------------ | ------------ |
| Level A      | 高可制造性   |
| Level B      | 可制造性一般 |
| Level C      | 低可制造性   |

注意，两类分级之间无任何关联。
