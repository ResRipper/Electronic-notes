# 无源元件标准尺寸与值

## 贴片元件标准尺寸

无法准确界定具体执行的标准编号，但目前贴片电阻和贴片陶瓷电容通常使用以下尺寸：

| 编号  | 毫米      | 英寸          |
| ----- | --------- | ------------- |
| 01005 | 0.4 x 0.2 | 0.016 x 0.008 |
| 0201  | 0.6 x 0.3 | 0.02 x 0.01   |
| 0402  | 1.0 x 0.5 | 0.04 x 0.02   |
| 0603  | 1.5 x 0.8 | 0.06 x 0.03   |
| 0805  | 2.0 x 1.3 | 0.08 x 0.05   |
| 1206  | 3.0 x 1.5 | 0.12 x 0.06   |
| 1210  | 3.2 x 2.5 | 0.125 x 0.10  |
| 1806  | 4.5 x 1.6 | 0.18 x 0.06   |
| 1812  | 4.6 x 3.0 | 0.18 x 0.125  |
| 1825  | 4.5 x 6.4 | 0.18 x 0.25   |
| 2010  | 5.0 x 2.5 | 0.20 x 0.10   |
| 2512  | 6.3 x 3.2 | 0.25 x 0.125  |
| 2920  | 7.4 x 5.1 | 0.29 x 0.20   |

### 尺寸建议

一般来说，更小尺寸的贴片元件有以下优势：

- 价格更低
- 占用空间更小

但是会存在以下问题：

- 可用的耐压及取值范围更低
- 难以拾取，容易丢失

手工贴片的建议最小尺寸为 `0603`，更小的尺寸使用会非常困难。

对于自动贴片，绝大部分贴片机都支持使用 `0402` 尺寸的元件，部分贴片机支持 `0201` 尺寸。因此，为在保证兼容性的前提下减小面积占用，建议最小使用 `0402` 尺寸的元件，并参考 SMT 代工厂的文档以确保可制造性。

## 元件值标准

无源元件的值通常满足 IEC 60063 中定义的 E 代码，以方便批量生产。E 代码被分为多个组，分别为

| 代码 | 最大误差 |
| ---- | -------- |
| E3   | 40%      |
| E6   | 20%      |
| E12  | 10%      |
| E24  | 5%       |
| E48  | 2%       |
| E96  | 1%       |
| E192 | 0.5%     |

其中仅 E192 包含的数值不常用。每个组包含的值被定义为：

$$
Value_{n}\approx\sqrt[m]{10^{n}}
$$

- $m$：E 系列的组编号，如 E3 的编号为 3
- $n$：$0, 1, 2 ... m-1$

对于 E3 ~ E24，结果需四舍五入到两位有效数字，其它的需四舍五入到三位有效数字。
